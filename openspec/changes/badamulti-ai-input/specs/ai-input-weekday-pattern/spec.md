## ADDED Requirements

### Requirement: 요일 및 날짜 범위 표현 파싱
`parseMultiEntries` 시스템 프롬프트는 오늘 날짜와 오늘 요일을 함께 전달해야 하며, AI는 요일 표현("월수금에", "이번주 내내")을 절대 날짜(YYYY-MM-DD)로 변환하여 각각 별도 항목으로 반환해야 한다.

#### Scenario: 단일 요일 표현
- **WHEN** 발화에 "월요일에 언어치료 갔어"가 포함되고 오늘이 목요일인 경우
- **THEN** 이번 주(또는 가장 최근 지난) 월요일 날짜로 1개 항목 반환

#### Scenario: 복수 요일 열거
- **WHEN** 발화에 "이번주 월수금에 ABA치료 다 갔어"가 포함된 경우
- **THEN** 해당 월·수·금 각각의 날짜로 3개 항목 반환, text에는 "ABA치료 갔음" 동일 내용

#### Scenario: 범위 표현
- **WHEN** 발화에 "이번주 내내 약 먹었어"가 포함된 경우
- **THEN** 월~오늘(또는 월~일) 날짜 각각으로 항목 반환, text에는 "약 복용함" 동일 내용

### Requirement: 미래 날짜 생성 금지
파싱 결과에 오늘 이후의 날짜가 포함되어서는 안 된다.

#### Scenario: 오늘 이전 날짜만 반환
- **WHEN** 발화에 "이번주 금요일에 치료"가 포함되고 오늘이 수요일인 경우
- **THEN** 이번 주 금요일이 아직 지나지 않았으므로 지난 주 금요일 날짜로 반환
