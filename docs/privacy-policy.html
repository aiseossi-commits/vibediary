개인정보 처리방침

최종 수정일: 2026년 4월 1일

바다(이하 "앱")는 음성으로 기록하는 관찰일지 서비스입니다. 본 방침은 앱이 수집하는 정보, 사용 방식, 보관 방법에 대해 설명합니다.

1. 수집하는 정보

  • 음성 파일: 녹음 버튼 사용 시 기기에 임시 저장되며, 텍스트 변환 후 삭제됩니다.
  • 기록 텍스트: STT 변환된 텍스트 및 사용자가 직접 입력한 내용
  • 관찰 대상 프로필 정보: 사용자가 등록한 이름(별칭)

앱은 이메일, 전화번호, 위치정보 등 식별 가능한 개인정보를 수집하지 않습니다.

2. 정보의 사용 목적

  • 음성 → 텍스트 변환(STT) 처리
  • AI 요약 및 구조화 (Google Gemini)
  • 기록 검색을 위한 임베딩 생성
  • 앱 기능 제공 및 오프라인 데이터 관리

3. 데이터 저장 위치

  • 음성 파일: 기기 로컬에만 저장. 외부 서버로 전송되지 않습니다.
  • 기록 데이터: 기기 내 SQLite 데이터베이스에 저장됩니다.
  • STT 텍스트 / AI 처리: Google Gemini 및 Groq Whisper API를 통해 처리되며, 처리 후 외부 서버에 저장되지 않습니다.

4. 제3자 서비스

앱은 다음 외부 서비스를 사용합니다:

  • Groq Whisper: 음성 → 텍스트 변환 (Groq 개인정보 처리방침)
  • Google Gemini: AI 요약 및 임베딩 생성 (Google 개인정보 처리방침)

모든 API 호출은 앱 자체 프록시 서버를 경유하며, 클라이언트에서 외부 API 키에 직접 접근하지 않습니다.

5. 데이터 보관 및 삭제

모든 데이터는 사용자의 기기에 보관됩니다. 앱을 삭제하면 기기 내 모든 데이터가 함께 삭제됩니다. 백업 파일은 사용자가 직접 내보내기 한 경우에만 생성됩니다.

6. 관찰 대상 정보

앱은 사용자가 지정한 관찰 대상(아이, 가족, 반려동물 등)의 기록을 다룹니다. 관찰 대상 정보(이름, 행동 기록 등)는 사용자가 직접 입력하며, 외부로 전송되거나 제3자에게 제공되지 않습니다.

7. 권한 사용

  • 마이크: 음성 녹음에 사용됩니다.
  • 음성 인식: iOS 기기 내장 STT에 사용됩니다.
  • 파일 접근: 백업 파일 내보내기/가져오기에 사용됩니다.

8. 방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이 페이지를 통해 안내합니다. 지속적인 앱 사용은 변경된 방침에 동의하는 것으로 간주됩니다.
