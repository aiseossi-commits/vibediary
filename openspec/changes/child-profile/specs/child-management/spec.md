## ADDED Requirements

### Requirement: 아이 프로필 등록

사용자는 이름으로 아이 프로필을 등록할 수 있어야 한다(MUST). 등록 즉시 활성 아이로 전환되어야 한다(SHALL).

#### Scenario: 아이 등록 성공

- **WHEN** 사용자가 아이 이름을 입력하고 등록을 확인한다
- **THEN** 새 아이 프로필이 생성된다
- **AND** 해당 아이가 활성 아이로 전환된다

#### Scenario: 이름 없이 등록 시도

- **WHEN** 사용자가 이름을 입력하지 않고 등록을 시도한다
- **THEN** 등록이 거부된다

### Requirement: 아이 프로필 수정 및 삭제

사용자는 등록된 아이의 이름을 수정하거나 프로필을 삭제할 수 있어야 한다(MUST).

#### Scenario: 이름 수정

- **WHEN** 사용자가 아이 이름을 변경하고 저장한다
- **THEN** 변경된 이름이 홈 화면 헤더에 즉시 반영된다

#### Scenario: 프로필 삭제

- **WHEN** 사용자가 아이 프로필을 삭제한다
- **THEN** 해당 아이의 기록은 삭제되지 않고 미지정(child_id = NULL) 상태로 유지된다
- **AND** 활성 아이가 없을 경우 홈 화면 제목은 "바다"로 표시된다

### Requirement: 활성 아이 전환

등록된 아이가 2명 이상일 때 사용자는 활성 아이를 전환할 수 있어야 한다(MUST). 전환 시 해당 아이의 기록만 표시되어야 한다(SHALL).

#### Scenario: 활성 아이 전환

- **WHEN** 사용자가 다른 아이를 선택한다
- **THEN** 홈 화면 제목이 "[이름]의 바다"로 변경된다
- **AND** 해당 아이의 기록만 목록에 표시된다

#### Scenario: 아이 없음 상태

- **WHEN** 등록된 아이가 없다
- **THEN** 홈 화면 제목은 "바다"로 표시된다
- **AND** 기록 필터링 없이 모든 기록이 표시된다
