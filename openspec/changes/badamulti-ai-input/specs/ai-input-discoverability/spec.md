## ADDED Requirements

### Requirement: 진주 버튼 하단 힌트 레이블 상시 표시
진주 버튼 하단에 "길게 눌러서 AI 입력" 힌트 텍스트가 항상 표시되어야 한다. 녹음 중·처리 중 상태에서는 숨겨진다.

#### Scenario: 일반 상태에서 힌트 표시
- **WHEN** 인라인 녹음이나 처리가 진행 중이지 않은 경우
- **THEN** 진주 버튼 아래 "길게 눌러서 AI 입력" 레이블 표시

#### Scenario: 녹음 중 힌트 숨김
- **WHEN** 인라인 녹음이 활성화된 경우
- **THEN** 힌트 레이블 숨겨지고 타이머와 AI 입력 모드 레이블 표시

#### Scenario: 처리 중 힌트 숨김
- **WHEN** STT·AI 처리가 진행 중인 경우
- **THEN** 힌트 레이블 숨겨지고 "기록중..." 표시
