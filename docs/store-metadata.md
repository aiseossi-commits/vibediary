■ 주요 기능

[ 음성으로 기록 ]
버튼 하나, 말 한마디. 녹음이 끝나면 텍스트 변환과 AI 요약이 자동으로 완성됩니다.

[ AI 자동 정리 ]
"음", "그니까" 같은 필러는 걸러내고, 체온·투약 같은 중요한 정보는 따로 구조화합니다. 말의 뜻은 절대 바꾸지 않습니다.

[ 태그 자동 분류 ]
#의료 #투약 #행동 #일상 #치료 — 기록 내용에 맞는 태그를 AI가 자동으로 붙입니다. 나중에 태그별로 모아볼 수 있습니다.

[ 캘린더 뷰 ]
월별 달력에서 기록이 있는 날을 한눈에 확인합니다. 날짜를 눌러 텍스트로 직접 추가할 수도 있습니다.

[ AI 등대 — 자연어 검색 ]
"지난달에 열 났던 게 언제였지?" 말로 물어보면 관련 기록을 찾아 답해줍니다.

[ 백업 & 복원 ]
전체 기록을 JSON 파일로 내보낼 수 있습니다. 폰을 바꿀 때 카카오톡 "나에게 보내기"로 파일을 옮긴 뒤, 새 폰에서 바다로 열면 바로 복원됩니다.
