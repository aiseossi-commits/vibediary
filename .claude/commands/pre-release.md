# 릴리즈 전 체크리스트

iOS/Android 빌드 전 종합 점검을 수행한다. 아래 항목을 순서대로 확인하고 결과를 보고한다.

## 검사 항목

### 1. 코드 품질
- `npx tsc --noEmit` 실행 → 타입 에러 없어야 함
- 미커밋 변경사항 확인 (`git status`) → 있으면 목록 표시

### 2. 버전 일관성
- `app.json`의 version, ios.buildNumber, android.versionCode 확인
- 마지막 커밋 이후 버전이 변경됐는지 확인

### 3. STATE.md & HQ.md 동기화
- `STATE.md` 마지막 업데이트가 최근 커밋과 일치하는가?
- `~/Documents/Obsidian Vault/바다 세계관/00_HQ.md`의 서비스 현황 테이블이 최신인가?

### 4. 환경변수
- `.env` 파일에 `EXPO_PUBLIC_WORKER_URL`, `EXPO_PUBLIC_WORKER_SECRET`이 설정되어 있는가?
- API 키가 코드에 하드코딩되어 있지 않은가? (`src/` 내부 grep)

### 5. DB 마이그레이션
- 최근 DB 버전 변경이 있었다면 `/check-db-migration` 결과 포함

### 6. 주요 기능 회귀 체크 (수동 확인 필요 항목 목록)
- 녹음 → STT → AI 요약 → 저장 플로우
- 오프라인 상태에서 앱 실행
- 아이 전환 시 데이터 분리
