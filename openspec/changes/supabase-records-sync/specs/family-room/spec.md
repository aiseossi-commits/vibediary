## MODIFIED Requirements

### Requirement: 가족 공유 UI (설정 화면)
설정 화면에 "가족 공유" 섹션이 있어야 하며(SHALL), 여기서 방 생성, 코드 입력, 현재 참여 상태를 확인할 수 있어야 한다(SHALL). 가족방에 참여 중인 경우 "함께 보기" 버튼을 통해 FamilyFeedScreen으로 이동할 수 있어야 한다(SHALL).

#### Scenario: 가족 공유 미설정 상태
- **WHEN** 사용자가 가족 공유에 참여하지 않은 상태에서 설정을 열 때
- **THEN** "가족방 만들기"와 "초대코드로 참여" 두 가지 옵션을 표시한다

#### Scenario: 가족방 참여 상태 확인
- **WHEN** 사용자가 가족방에 참여한 상태에서 설정을 열 때
- **THEN** 현재 초대코드, 멤버 수를 표시하고 "함께 보기" 버튼과 "가족방 나가기" 옵션을 제공한다

#### Scenario: 함께 보기 진입
- **WHEN** 가족방 참여 상태의 사용자가 "함께 보기" 버튼을 탭할 때
- **THEN** FamilyFeedScreen으로 이동하여 가족 멤버 전체의 기록을 조회한다
