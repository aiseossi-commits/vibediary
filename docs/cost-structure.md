# VibeDiary 비용 구조 설명서

> 대상: API 과금에 대해 처음 접하는 분

---

## 1. 앱이 돈을 쓰는 이유

VibeDiary는 사용자가 말하면 자동으로 텍스트로 변환하고, AI가 요약/정리해주는 앱입니다.
이 두 가지 기능이 외부 AI 서비스를 사용하기 때문에 이용할 때마다 비용이 발생합니다.

```
사용자 녹음
  → (1) STT: 음성을 텍스트로 변환  ← Groq Whisper API 사용
  → (2) AI: 텍스트를 요약/태그 생성 ← Google Gemini API 사용
  → 앱에 저장
```

---

## 2. STT 과금 원리 (음성 → 텍스트)

**사용 서비스**: Groq Whisper Large v3 Turbo
**과금 방식**: 녹음된 오디오 시간만큼 과금 ("들은 만큼 낸다")

| 단위 | 가격 |
|------|------|
| 1시간 오디오 | $0.04 (약 58원) |
| 1분 오디오 | $0.00067 (약 0.97원) |
| 15초 오디오 | $0.000167 (약 0.24원) |
| 30초 오디오 | $0.000333 (약 0.48원) |

> **예시**: 사용자가 20초짜리 녹음을 하면 약 0.32원이 청구됩니다.

**참고 (교체 전 비용)**: 기존에 사용하던 OpenAI Whisper는 동일 조건에서 약 9배 비쌌습니다.
→ Groq로 교체 후 STT 비용 약 89% 절감

## 3. AI 과금 원리 (텍스트 요약/정리)

**사용 서비스**: Google Gemini 2.5 Flash Lite
**과금 방식**: 주고받은 텍스트 양(토큰)만큼 과금

### 토큰이란?
텍스트를 쪼갠 단위입니다. 대략 한국어 1글자 ≈ 1~2토큰으로 이해하면 됩니다.

| 방향 | 가격 |
|------|------|
| 입력 (앱 → AI) | $0.10 / 100만 토큰 |
| 출력 (AI → 앱) | $0.40 / 100만 토큰 |

### 녹음 1회당 AI 비용 계산
- 입력: 시스템 프롬프트 + 녹음 텍스트 ≈ 500토큰
- 출력: 요약 + 태그 + 구조화 데이터 ≈ 300토큰
- **1회 비용**: 약 $0.00015 (약 0.22원)

> 녹음 길이와 무관하게 거의 고정 비용입니다. (텍스트 양이 크게 달라지지 않으므로)

## 6. 비용이 발생하지 않는 경우

- **iOS 네이티브 STT 사용 시**: iPhone에 내장된 음성인식이 먼저 동작하며, 이 경우 Groq STT 비용이 발생하지 않습니다. Groq는 iOS 네이티브 STT 실패 시 fallback으로만 동작합니다.
- **AI 처리 실패 시**: 네트워크 오류 등으로 AI 처리가 실패하면 오프라인 큐에 저장되었다가 나중에 재처리됩니다. 이때 비용은 재처리 시점에 발생합니다.
